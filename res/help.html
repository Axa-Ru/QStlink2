This application can be used through the GUI or via the command line. (cli)

The GUI is pretty intuitive, you should no be able to do anything wrong.

Command line usage:
cli: Enable command line mode
flash:[read|write]:[noerase]:PATH = Manual dumping/flashing of the device. You can choose not to erase the flash prior writing.
debug: Enable debug output

Example to flash the device:
sudo ./qstlink2 flash:write:noerase:/home/me/myfile.bin
